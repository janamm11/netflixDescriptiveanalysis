Netflix Data: Cleaning, Analysis, and Visualization
A data analysis project exploring Netflix's content library from 2008 to 2021. This project leverages Python to clean, analyze, and visualize trends in Netflix's dataset uncovering patterns in content distribution, popular genres, and release trends across more than a decade of streaming history.
Overview
Streaming platforms generate massive amounts of content data that, when analyzed effectively, reveal meaningful insights about industry trends and audience preferences. This project walks through the full data analysis pipeline ,from raw data cleaning to visual storytelling ,using some of Python's most powerful data science libraries.
Features

Data Cleaning : Handles missing values, duplicates, and inconsistencies to ensure reliable analysis
Exploratory Data Analysis : Uncovers patterns in content type, genre popularity, and release behavior over time
Data Visualization : Presents findings through clear, informative charts built with Seaborn and Matplotlib

Tech Stack

Python
Pandas & NumPy : data manipulation and cleaning
Matplotlib & Seaborn :data visualization
